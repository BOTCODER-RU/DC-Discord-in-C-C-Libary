# DC — Discord in C

![Language](https://img.shields.io/badge/language-C-blue)
![Build](https://img.shields.io/badge/build-CMake-green)
![License](https://img.shields.io/badge/license-MIT-orange)

Библиотека `DC` (Discord in C) — это библиотека, разработанная на языке программирования C для взаимодействия с Discord API. Основная цель проекта — предоставить простой и достаточно низкоуровневый инструмент для создания Discord-ботов и других приложений, использующих возможности Discord напрямую из C.

Библиотека использует HTTP-запросы, обработку JSON и взаимодействие с Discord API через Apache / libcurl и другие вспомогательные компоненты.

---

# Возможности

* Подключение к Discord API.
* Отправка и получение HTTP-запросов.
* Работа с сообщениями, каналами и серверами.
* Отправка сообщений от имени бота.
* Получение информации о пользователях, серверах и каналах.
* Поддержка JSON.
* Возможность расширения под собственные задачи.
* Простая интеграция в проекты на C.

# Подключение библиотеки

Для подключения библиотеки используйте основной заголовочный файл:

```c
#include <discordc/discord.h>
```

Пример создания клиента:

```c
#include <discordc/discord.h>

int main(void)
{
    DiscordClient client;

    discord_client_init(&client, "YOUR_BOT_TOKEN");

    discord_send_message(
        &client,
        "CHANNEL_ID",
        "Hello from DC library!"
    );

    discord_client_cleanup(&client);
    return 0;
}
```
